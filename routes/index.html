Quick Start

  1. Install the npm dependecy

    npm i exjson

  2. Create a index.json file in the root of your proyect
  3. Write this on the file created in the previos step 
    "models":{  
	"user":{  
		"id":"serial-PK",  
		"username":"string",  
		"password":"string"  
	}  
}

That is all!! You already created a model user with an id, an username and a password

Tutorial

ExJson is a express framework that allow you to create a rest api easier and faster using jsons files

How it works

For understand how it works you can see this graph.

  1. First the user wirte a json file like “index.json”.
  2. Express read this file and transform his data in endpoints and postgres entities.
  3. It generate a functional rest api.

Create models

The models are created like:

"models":{  
	"model_name":{  
		"id":"datatype-PK",  
		"attribute":"datatype"  
	}  
}

Let see an example:

"models":{  
	"user":{  
		"id":"serial-PK",  
		"username":"string",  
		"password":"string"  
	}  
}

Here we created a “user” model with an id, an username and a password.

Data Types

By default the datatype is string, but there are other types like:

Type Description
boolean true or false
int number until 64 bits
serial auto increment int
string chain of characters
